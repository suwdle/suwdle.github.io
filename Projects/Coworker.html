Project Overview

AI Recommendation & Financial Analysis Chatbot for SMEs

Coworker helps small and medium businesses quickly discover suitable financial products and understand their own financial statements. It combines a hybrid recommendation engine (semantic similarity + keyword matching) with a RAG-based chatbot that explains key ratios in plain language. :contentReference[oaicite:3]{index=3} :contentReference[oaicite:4]{index=4}

Features

  • Hybrid Product Ranking: We generate a company-specific query and keywords from profile data, then score products using cosine similarity over embeddings plus keyword match signals. Top candidates are returned to the user. :contentReference[oaicite:5]{index=5}
  • Embedding Fine-Tuning: A synthetic triplet dataset (anchor/positive/negative) trains the embedding model via Triplet Loss, improving separation (Avg. Pos 0.72 vs 0.61; Avg. Neg 0.06 vs 0.50; margin â†‘ to 0.66). :contentReference[oaicite:6]{index=6}
  • RAG Financial Chatbot: Users upload statements; the agent retrieves top-k evidence chunks and generates verified answers for metrics such as liquidity and debt ratios. :contentReference[oaicite:7]{index=7}

Python, Scikit-learn, LangGraph, Vector Embeddings (Hybrid Search), Docker
