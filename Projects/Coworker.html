Project Overview

RAG Solution for Small and Medium Enterprises

This system analyzes corporate financial management documents and recommends appropriate support programs. It also retrieves real-time information through external APIs to provide diverse and accurate descriptions of comparisons with other companies and current industry conditions.

Features

  • Developed using LangGraph, this solution goes beyond simple RAG by enabling the use of appropriate tools based on user intent.
  • Tasks are distributed among different agents, breaking down complex work into smaller, manageable components.
  • The system provides feedback on responses to fill in missing information and deliver more accurate answers.

Technologies Used

Python, LangGraph, Docker, AWS Bucket
